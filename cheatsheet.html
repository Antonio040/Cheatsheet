Here is a list of the most important HTML tags and CSS properties:

use em for italic and strong for bold
to create a line break use the "br" tag
to create a horizontal line use the "hr" tag
to create a link use the "a" tag
to create a list use the "ul" tag for unordered list and "ol" tag for ordered list
to create a list item use the "li" tag
to create a table use the "table" tag
to create a table row use the "tr" tag
to create a table header use the "th" tag
to create a table data use the "td" tag
to create a form use the "form" tag
to create a button use the "button" tag
